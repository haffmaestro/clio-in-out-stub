== README

Ruby Version 2.1.5

Installation:


1. $ git clone 
2. $ bundle install
3. $ brew install redis
4. $ rake db:create db:migrate db:seed
5. $ redis-server /usr/local/etc/redis.conf
6. $ rails s


Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
